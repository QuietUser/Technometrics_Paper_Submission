Hi. Thanks for taking a look at this.

You will find the following in this GitHub repository:


Plots Folder: This folder contains the code for all of the plots in the paper.  It pulls from the results (.Rda) files in the other folders.

Data Folder:  This folder contains the raw water filter data, along with the separated cycles. 

Simulation_Study Folder: This folder contains the code for the simulation study as well as the results of the simulation study for each method.

Water_Filtration_Application Folder: This folder contains the code for the analysis of each headloss cycle for each method as well as the results.

Functions Folder:  This folder contains the functions used in the siumlation study and the headloss analysis.
