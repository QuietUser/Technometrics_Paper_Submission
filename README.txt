Hi. Thanks for taking a look at this.

You will find the following in this github repository:



Data Folder:  This folder contains the raw water filter data, along with the separated headloss cycles.

Plots Folder: This folder contains the code for all of the plots in the paper.

Simulation_Study Folder: This folder contains the code for the simulation study in a markdown as well as the results of the simulation study for each method.

Water_Filtration_Application Folder: This folder contains the code for the simulation study, as a single .Rda or as individual .R scripts.  The results are also stored here.

Functions Folder:  This folder contains the functions used in the siumlation study and the headloss analysis.
